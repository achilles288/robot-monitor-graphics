Detailed Description

Maintains the image data before context startup.

During the font loading function, it translate the file to a chunk of data and they are desired to be loaded into GPU for shader computations. However this can't be done before the OpenGL context shows up. So, this temporary storage class is made to maintain the data for a while. before the context startup.
