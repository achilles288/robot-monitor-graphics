Detailed Description

2D/3D OpenGL graphics context

Manages 2D/3D objects with GPU context resources and shader programs. Abstract layer of OpenGL context.

◆ setDirectionalLightAngles() [1/2]

void rmg::Context::setDirectionalLightAngles ( float  pitch,
float  yaw 
)

Sets the directional lighting angles.

Directional light takes the main role in fragment shaders and shadow mappings for 3D object viewing. Rotation order is ZY (Yaw-Pitch). Rolling has no effect. Directional light vector with angle values of zeros is pointed towards the Y-axis which is the same as initial camera direction.

Parameters
pitchAngle above xy-plane
yawAngle about z-axis
