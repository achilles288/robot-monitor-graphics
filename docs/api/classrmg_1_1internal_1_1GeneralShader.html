Detailed Description

Peforms general tasks like positioning and lighting.

Operates MVP matricies through the list of 3D objects. Reads the appearance model of the objects and display them on screen processing in the general fragment shader. Positioning is done by processing MVP (Model-View-Projection) matricies in vertex shader.
