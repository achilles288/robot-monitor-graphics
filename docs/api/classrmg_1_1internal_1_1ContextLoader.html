Detailed Description

Loads data in memory into GPU.

After preprocessing of 3D data like model reading, texture reading, the parsed data are temporarily stored in a queue. When the OpenGL context starts up, these data are loaded into the GPU, the memory data are deleted and the newly obtained GPU resource is later used for rendering models frequently.
