Detailed Description

3D object whose model and appearance can be controlled quickly

The constructor builds a vertex buffer object to load into the GPU and keeps the address to that resource. The model matrix and material properties like color, diffusion and specularity coefficient are passed into shader programs to render the 3D object. Mainpulation of position, rotation and scale keep up with the model matrix.
